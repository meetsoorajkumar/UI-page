How the greedy logic works
1️⃣
Scan even floors
Check floors 2 → 4 → 6 for any free slot. Stop at the first one found.
2️⃣
Fall back to odd
Only if all even floors are full, scan floors 1 → 3 → 5.
3️⃣
Assign & return
Link the slot to the driver's ticket. If completely full, reject the request.
Pseudocode — directly from the paper
Step 1 — find a parking slot (greedy selection)
// Scan floors that have an elevator (even floors) first
for (i = 0; i < floorHaveElevator; i++) {
  if (parking[i] == null) {
    return parking[i]; // first free slot found → done
  }
}
return null; // no even floor slot found → try odd floors next
Step 2 — assign driver to the slot
// Write driver data into the selected parking slot
for (i = 0; i < sizeParkir; i++) {
  if (parking[i].tiketID == tiketID) {
    set parking[i] = driver; // bind slot → driver
    return;
  }
}
return null;
Step 3 — exit: calculate fee & free the slot
// Driver exits; compute price and reset the slot to available
for (i = 0; i < sizeParkir; i++) {
  if (parking[i].tiketID == tiketID) {
    calculateParkingPrice();
    // price = (timeOut − timeIn) × ratePerMinute
    setParkingAreaToAvailable(); // slot.tiketID = NULL
  }
}
